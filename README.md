# naver-downloader

Python script to download Naver Webtoon episodes with:
- Stable download
- Browser-level image quality
- Support for lazy loading

## Requirements
- Python 3.9+

## Install dependencies
pip install aiohttp httpx requests beautifulsoup4

## Usage
python nava.py COMIC_ID START END OUTPUT_PATH

Example:
python nava.py 816614 1 10 ./webtoons
